# 🔴 Atomic Red Team Capstone Project

A collaborative red team exercise completed with three classmates as part of the Fullstack Academy Cybersecurity Analytics Bootcamp.

**Presented by:** Claudia Jarmakowicz & Team  
**Attack Technique:** T1547.001 — Boot or Logon Autostart Execution: Registry Run Keys / Startup Folder

---

## 📋 Overview

This project simulates a real-world adversary attack using the Atomic Red Team framework. Each team member selected and executed a different MITRE ATT&CK technique, documenting the full attack lifecycle from setup through detection and mitigation.

**My presentation covers:**
- Environment setup
- What T1547.001 is and how it works
- Splunk and BlueSpawn detection
- Checking the Winlogon Userinit registry key
- T1547.001 Tactic, Technique & Sub-technique breakdown
- Mitigation techniques
